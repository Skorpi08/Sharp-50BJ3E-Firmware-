1. Verwenden Sie einen USB-Stick, der im **FAT32-Format** formatiert ist.
2. Kopieren Sie die **Binärdatei** und die Datei **UMC_KMODE.txt** in das Hauptverzeichnis (Root) des USB-Sticks.
3. Wenn der Fernseher vom Stromnetz getrennt ist, stecken Sie den USB-Stick in den USB-Anschluss des Fernsehers und schließen Sie das Gerät wieder an das Stromnetz an.
4. Warten Sie, bis der Upgrade-Vorgang abgeschlossen ist. Achten Sie auf den Fortschrittsbalken und auf die Blinkfrequenz der TV-LED. Wenn der Fortschrittsbalken 100 % erreicht und die LED schneller blinkt, ist das Upgrade abgeschlossen.
5. Führen Sie einen **Aus-/Einschaltzyklus** durch, während der USB-Stick weiterhin im Fernseher eingesteckt bleibt, und lassen Sie den Fernseher vollständig starten. In diesem Stadium können Sie ein **doppeltes Bild** oder **ein fehlerhaftes Bild** sehen – das ist normal.
6. Trennen Sie den Fernseher wieder vom Stromnetz und **entfernen Sie den USB-Stick**.
7. Schließen Sie das Gerät erneut an das Stromnetz an und lassen Sie den Fernseher vollständig starten. An dieser Stelle sollte das Bild korrekt angezeigt werden, und das Upgrade ist abgeschlossen.

**Hinweis:**
In den Schritten 5 und 7, wenn Sie den Fernseher vom Stromnetz trennen, prüfen Sie bitte die TV-LED. In manchen Fällen bleibt sie noch einige Sekunden lang eingeschaltet. Bitte warten Sie, bis die LED vollständig **ausgeschaltet** ist, bevor Sie den Strom wieder anschließen.
